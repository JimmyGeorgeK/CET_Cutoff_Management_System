# CET_Cutoff_Management_System
The CET CutOff Management System is developed to override the problems prevailing
in the practicing manual system, this software is supported to eliminate and in some
cases reduce the hardships faced by this existing system. Moreover this system is
designed for the particular need of the company to carry out operations in a smooth
and effective manner. The application is reduced as much as possible to avoid errors
while entering the data. It also provides error message while entering invalid data. No
formal knowledge is needed for the user to use this system. Thus by this all it proves
it is user-friendly. CET CutOff Management System, as described above, can lead
to error free, secure, reliable and fast management system. It can assist the user to
concentrate on their other activities rather to concentrate on the record keeping. Thus
it will help organization in better utilization of resources. Every organization, whether
big or small, has challenges to overcome and managing the information of Students,
Registrations, Courses at the management level. This is designed to assist in indexing
and will help you ensure that your organization is equipped with the right level of
information and details for your future goals. Also, for those busy executive who are
always on the go, our systems come with remote access features, which will allow you
to manage your workforce anytime, at all times. These systems will ultimately allow
you to better manage resources.

# Application
This project is developed in favour of Students where all the details of the institutions
will be stored by Admin. It helps to overcome the problem that arises while handling
the information and details Manually. The project is built in such a way that future
modification can be easily done. It gives appropriate access to the authorized user i.e
Admin. Accessing, updating all the operations are done at ease.
The operations of the system is done using the File Structure concepts. In this
project we use C++ code. The main Goal of the project is to achieve the efficient
and effective system for the Cet Cut Off Management. The future implementation of
the project includes the advanced Technologies to increase efficiency and effectiveness
of the system and to increase simplicity of the project by including available modern
techniques
